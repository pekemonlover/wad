# AWS App Deployment

## Theory
Amazon Web Services (AWS) provides scalable cloud computing solutions. This project represents a simple app that can be deployed to an AWS service like EC2, S3 (static hosting), or AWS Amplify.

## Prerequisites
- AWS CLI (optional, for deployment).

## Setup and Run
1. Open `index.html` locally in a browser to test.
2. Upload to S3 or deploy via your preferred AWS pipeline.
